# 브랜드 네이밍 후보 검토_한솔

| 후보 | 의미 | 느낌·톤 | 충돌·등록 상태 |
|---|---|---|---|
| **안아** | 安兒(안전한 아이) + 안아주다 | 따뜻·안전, 직관적 | ❌ 35류 S2090에 '안아' 선등록(여성복 브랜드, 실사용) → 등록·사용 불가, 참고용만 |
| **리피 (Leapy)** | leap = 발달 도약 | 발랄·성장 응원 | ⚠️ 국내 동종 앱 없음(확인). 단 영문 Leap·Leaply·Wonder Weeks와 칭호유사 심사 가능 → KIPRIS 확인 |
| **포미 (POMI)** | for me(내 아이 위한) + Picked·Only-safe·Mom-trusted | 따뜻 + 큐레이션 | ⚠️ 9·35류는 비교적 깨끗 가능. 단 식품 'Pomì' 상표 있어 이유식·식품 PB면 5·29·30류 충돌 확인 |
| **플릿링 (Fleetling)** | fleeting(찰나) + ling(어린것) | 시적·정서적, 글로벌 | ✅ 웹상 깨끗. 영어라 일반 사용자에 뜻이 바로 안 읽힘(태그라인 보완 필요) |
| **순리 (Soonly)** | so soon + 순리(順理) 이중의미 | 정서 + 위트, 국내 친화 | ✅ 웹상 깨끗. 한글로도 읽히고 의미 통함 |
