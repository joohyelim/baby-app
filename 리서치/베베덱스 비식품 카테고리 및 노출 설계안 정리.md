# 맘가이드 앱 분석 기반, 베베덱스 비식품 카테고리 및 노출 설계안
---

## 1. 카테고리 구분

| 단계 | 구분 | 예시 |
|---|---|---|
| 1단 홈 타일 | 육아 장면 (부모가 하는 일) | 씻기기, 먹이기, 기저귀 갈기 |
| 2단 목록 칩 | 품목 종류 | 크림 / 로션 / 오일 |
| 3단 필터 | 속성 | 안심 상태, 무향, KC 인증 |

**홈 타일 9개**

| 장면 타일 | 품목 칩 |
|---|---|
| 🛁 씻기기 | 워시, 샴푸, 아기욕조 |
| 🧴 발라주기 | 크림, 로션, 오일, 선케어, 수딩젤, 태열크림 |
| 👶 기저귀 갈기 | 기저귀, 물티슈, 발진크림 |
| 🍼 먹이기 | 젖병, 젖병솔, 세정제, 빨대컵, 그릇과 식판, 스푼, 소독기 |
| 🦷 이 닦기 | 치약, 구강티슈, 치발기 |
| 🫧 빨래·청소 | 세탁세제, 섬유유연제, 유아 클리너 |
| 👕 입히기·재우기 | 배냇저고리, 속싸개와 손싸개, 가제, 침구 |
| 🌡️ 아플 때 | 체온계, 콧물흡입기, 습도계 |
| 🧸 놀아주기 | 모빌, 원목완구, 촉감책과 보드북, 치발기 |

**보완 장치 3가지**

- 데이터가 아직 없는 타일은 숨기지 않고 '곧 열려요' 표기. 화면 골격은 처음부터 완결하고 데이터는 준비되는 대로 채움
- 시기별 큐레이션: 시기에 따라 카테고리에 바꿀지 결정 필요

## 2. 왜 장면 기준인가

- 맘가이드 홈은 용도 언어(유아 보습, 씻기)와 품목 언어(물티슈, 세제)가 섞여 있음.
- 장면 기준이면 배치 기준이 하나로 정리: '이 물건으로 부모가 뭘 하는가'. 신규 품목이 생겨도 배치 논쟁이 짧아짐
- 부수 효과: 부모에게 한 일거리인 물건들이 한 타일에 모임. 먹이기 = 젖병(식기) + 세정제(생활화학) + 소독기(기구), 내부 분류가 달라도 노출에서 재결합

**월령이 늘어나도 타일은 유지**

- 부모가 하는 일(씻기기, 먹이기, 입히기)은 월령이 늘어도 사라지지 않음. 바뀌는 것은 그 일에 쓰는 물건 → 타일이 아니라 칩 단위에서 교체(먹이기: 젖병 → 빨대컵 → 일반 식기). 품목 기준 분류였다면 월령 확장 때마다 홈 개편이 필요했을 구조
- 예외: 기저귀 갈기는 배변 훈련 완료 시 소멸하는 장면. 반대로 배변 훈련(유아변기, 훈련 팬티), 외출하기(카시트, 유모차) 같은 장면이 24개월 이후 등장 → 타일 추가·교체로 대응, 축은 그대로
- 월령 반영 방식: 아이 월령에 따라 타일 정렬과 기본 칩을 조정하고, 시기별 큐레이션이 월령 변화를 흡수. 현재 9개 타일은 0~24개월 기준 구성

## 3. 화면 동작 (요약)

- 홈: 월령 히어로 → 시기별 큐레이션 → 바로가기 → 장면 타일 9개
- 타일 진입: 상단 칩으로 품목 거름 + '안심만 보기' 토글
- 목록 카드: 품목 정보 성격에 따라 3종. 성분형(전성분 있는 세제·로션 = 점수), 소재형(젖병 = 재질 판정), 신뢰지표형(체온계·의류 = 인증 배지만)
- 필터: 월령, 품목 칩, 안심 상태(안심/확인 필요 2상태), 안심 태그(4절), 인증, 제조국

**제조국 노출 원칙 (중국산 민감도 대응)**

- 배경: 주 유저(엄마)의 중국산 거부감이 매우 큰 품목군. 제조국을 찾기 어렵게 두면 앱 신뢰 자체가 흔들림 → 숨기지 않고 확실하게 명시
- 카드: 제조국을 모든 카드에 상시 표시. 미표기 제품은 '제조국 미표기'로 노출해 미표기 자체가 신호가 되게 함
- 필터: 국산/수입 2분이 아니라 국가 단위(한국, 중국, 미국, 유럽 등)로 세분화해 걸러볼 수 있게 함
- 주의 1: 브랜드 국적 ≠ 제조국(국내 브랜드의 중국 OEM 다수). 표기는 제조국 기준
- 주의 2: 앱이 '중국산 = 나쁨'이라는 가치판단은 하지 않음. 점수와 등급에 제조국 미반영, 정보를 투명하게 보여주고 판단은 유저에게 → 명예훼손·공정거래 리스크 회피하면서 유저 니즈 충족

## 4. 안심 태그: 종류와 노출 방식

안심 태그 = 제품의 안심 특징을 한 단어로 요약한 라벨. 안심 기준 정리 5~6절의 ✓ 항목에서 추출. 제조사 표시와 공공 자료로 확인된 것만 부착하고, 표기가 없으면 태그를 붙이지 않음(미표기를 무첨가로 간주하지 않음)

**장면 타일별 태그 사전**

| 장면 타일 | 안심 태그 |
|---|---|
| 🛁 씻기기 / 🧴 발라주기 | 무파라벤, 무인공향, 무색소, CMIT/MIT 무첨가, 저자극 테스트 완료 |
| 👶 기저귀 갈기 | 무형광증백제, 무파라벤, CMIT/MIT 무첨가, 천연펄프/순면, 저자극 테스트 완료 |
| 🍼 먹이기 | BPA-free, 프탈레이트-free, 식품용 적합, 용출시험 통과, 안전소재(PPSU, 스테인리스, 강화유리) |
| 🦷 이 닦기 | 무파라벤, 무색소, 무인공향, CMIT/MIT 무첨가 |
| 🫧 빨래·청소 | 무형광증백제, CMIT/MIT 무첨가, 무인공향·색소, 안전확인 신고, 친환경 인증 |
| 👕 입히기·재우기 | 순면 100%, 무형광, OEKO-TEX Class I, GOTS, 유해물질 시험 통과 |
| 🌡️ 아플 때 | KC 인증, 전기 KC, 식품접촉부 BPA-free, 리콜 이력 없음 |
| 🧸 놀아주기 | KC 안전확인, 무프탈레이트, 친환경 도료(수성), 원목·천연소재 |

**앱 노출 3지점**

- 목록 필터: 지금 보는 타일에 해당하는 태그만 필터로 노출(씻기기 목록에서 BPA-free는 안 보임). 다중 선택 가능
- 제품 카드: 보유 태그 중 최대 2개만 ✓로 표시(카드 과밀 방지, 우선순위는 타일별 태그 사전의 나열 순서)
- 제품 상세: 보유 태그 전체 + 태그별 근거(제조사 표시인지, 공공 자료 확인인지) 표시

## 5. 근거 (요약)

- 내부 관리 분류는 노출과 별개로 유지: LIFE(성분), WARE(소재), GEAR·WEAR(인증), TOY 8개 중분류. 판정 기준은 법령 단위라 장면과 다르게 쪼개야 정확 (상세: 안심 기준 정리 3~6절)
- 맘가이드에서 가져온 것: 두 겹 분류(내부 관리 따로, 소비자 노출 따로), 등급과 인증 마크 분리 표기, '법적 기준 통과 + 아기 기준 한 번 더' 등급 설명 논리
- 맘가이드와 다르게 간 것: 홈 축 혼재를 물려받지 않고 장면 단일 축, 자체 심사 마크 없이 공적 인증(KC 등)만 배지로, 소비자 등급은 안심/확인 필요 2상태

---

## 6. 미결 사항

1. '곧 열려요' 프리뷰: 확장 기대 vs 빈 껍데기 신뢰 리스크. 디자이너와 A/B 논의
2. 장면 타일과 관리 분류는 다대다 매핑(치발기 = 이 닦기 + 놀아주기). 스키마 반영 필요
3. 타일 이름(씻기기, 발라주기)이 부모에게 직관적인지 출시 전 5인 내외 간이 테스트
4. 24개월 이후 신규 장면(배변 훈련, 외출하기)의 추가 시점과 월령별 타일 정렬 규칙
