## 1. 발달 단계 분류 기준 TOP 5

### ① 한국 국가 영유아 건강검진 (국민건강보험공단)

| 검진 회차 | 월령 | 주요 평가 영역 |
|---|---|---|
| 1차 | 생후 14일 | 신체계측, 청각·시각 이상 |
| 2차 | 4개월 | 대근육운동, 사회성 |
| 3차 | 9개월 | 소근육, 언어 발달 |
| 4차 | 18개월 | 보행, 언어(단어), 자조 |
| 5차 | 30개월 | 언어(문장), 사회성 |
| 6차 | 42개월 | 인지, 대소변 가리기 |
| 7차 | 54개월 | 집중력, 또래 관계 |
| 8차 | 66개월 | 취학 전 발달 종합 |

- **평가 도메인**: 대근육운동 / 소근육운동 / 인지 / 언어 / 사회성 / 자조 (6개 영역, 영역별 8문항)
- **판정 기준**: 상위 16%ile(빠른 발달) / 16~84%ile(또래 수준) / 2.3~15.9%ile(주의) / 하위 2.3%(심화평가)
- **플랫폼 활용 포인트**: 한국 부모에게 가장 친숙한 기준. 검진 월령(14일·4개월·9개월·18개월·30개월·42개월)을 제품 큐레이션 구간으로 그대로 활용 가능

### ② AAP (미국소아과학회) Well-Child Visit 스케줄

| 시기 | 방문 시점 |
|---|---|
| 신생아기 | 생후 3~5일, 1개월 |
| 영아기 | 2, 4, 6, 9, 12개월 |
| 걸음마기 | 15, 18, 24, 30개월 |
| 유아기 | 3, 4, 5세 (연 1회) |

- **발달 스크리닝**: 9개월·18개월·30개월에 정식 발달검사 / 18개월·24개월에 자폐 스크리닝
- **마일스톤 업데이트 (2022)**: 하위 75% 이상 아동이 달성하는 행동을 기준으로 재정의 → "대부분 아이가 이 시기에 이렇게 한다"는 접근
- **플랫폼 활용 포인트**: 글로벌 기준 중 가장 공신력 높음. 일본·북미 진출 시 기준으로 활용 가능

### ③ Piaget 인지발달 이론

| 단계 | 월령 | 핵심 발달 |
|---|---|---|
| 감각운동기 (Sensorimotor) | 0~24개월 | 오감+신체로 세상 탐색, 대상영속성(~8개월) 형성 |
| 전조작기 (Preoperational) | 2~7세 | 상징적 사고, 가작놀이(역할극), 언어 폭발적 성장 |

- **제품 연결**: 감각운동기 → 오감 자극 장난감(딸랑이, 치발기, 모빌), 전조작기 → 역할극 장난감, 그림책, 퍼즐
- **플랫폼 활용 포인트**: "지금 우리 아이의 두뇌가 이렇게 발달 중이에요" 식의 맥락 콘텐츠와 묶음 추천에 강력

---

### ④ WHO 운동발달 마일스톤 (Global Motor Development Standards)

| 마일스톤 | 평균 달성 월령 |
|---|---|
| 혼자 앉기 | 6개월 |
| 붙잡고 서기 | 8~9개월 |
| 기어다니기 (손무릎) | 9개월 |
| 붙잡고 걷기 | 10~11개월 |
| 혼자 서기 | 11~12개월 |
| 혼자 걷기 | 12~13개월 |

> WHO 기준은 **24개월 이내 달성을 정상 범주**로 보며, 저소득~고소득 6개국 데이터 기반
- **플랫폼 활용 포인트**: "우리 아이 지금 기어다니기 시작했어요" → 기어다니기 안전 관련 제품(매트, 가드, 범퍼침대) 즉시 큐레이션

### ⑤ Erikson 심리사회 발달 이론

| 단계 | 월령 | 핵심 과제 | 관련 육아용품 카테고리 |
|---|---|---|---|
| 신뢰 vs 불신 | 0~18개월 | 양육자와 안정 애착 형성 | 수면용품, 스킨케어, 수유용품 |
| 자율 vs 수치 | 18개월~3세 | 독립심, 대소변 가리기 | 변기, 자조 도구(숟가락·컵), 보행기 |
| 주도 vs 죄책감 | 3~6세 | 주도적 탐색, 상상놀이 | 역할극 세트, 미술도구, 야외놀이 |

- **플랫폼 활용 포인트**: "이 시기 아이에게 가장 중요한 것" 프레이밍으로 신뢰도 높은 추천 경험 설계

## 2. 기준별 마켓플레이스 유효 DIFFERENCE 분석

| 기준 | 단계 구분 방식 | 마켓플레이스 차별성 |
|---|---|---|
| 한국 건강검진 | 검진 월령 기반 (14일·4·9·18·30·42·54·66개월) | 한국 부모에게 가장 익숙. "검진 전 체크리스트 + 추천템" 패키지로 전환율 UP |
| AAP | 방문 시점 기반 (촘촘한 12개월까지) | 글로벌 확장 시 기준. 과학적 근거 강조 마케팅 |
| Piaget | 인지발달 단계 2분 (감각운동·전조작) | 장난감·교구 큐레이션에 특화. "두뇌발달" 프레임이 프리미엄 포지셔닝과 잘 맞음 |
| WHO 운동발달 | 운동 마일스톤 6단계 | 행동 트리거 기반 ("기기 시작했어요") 알림 연동에 최적 |
| Erikson | 심리사회 3단계 | 감성적 스토리텔링 ("신뢰를 쌓는 시기") → 콘텐츠 마케팅·브랜딩에 강점 |

**핵심 인사이트**: 단일 기준만 쓰는 플랫폼은 없음. 의료 기준(건강검진·AAP)으로 신뢰를 쌓고, 발달 이론(Piaget·Erikson)으로 감성 스토리를 입히는 이중 구조가 가장 효과적.

## 3. 육아용품 플랫폼 월령 큐레이션 제안 (혼합 기준)

### 큐레이션 3-Layer 프레임워크

> 기존 리서치는 L1·L2에 집중됐으나, **실제 구매 트리거는 L3(케어 루틴 전환)가 가장 강력**함.

| 레이어 | 기준 유형 | 예시 | 구매 트리거 강도 |
|---|---|---|---|
| **L1. 의료/검진** | 건강검진, AAP 방문 | 9개월 검진 | 낮음 — 간접적 |
| **L2. 발달 행동** | WHO 운동발달, Piaget | 기기 시작, 첫 걸음 | 중간 — 제품 필요 인식 |
| **L3. 케어 루틴 전환** | 소아과/영양 가이드라인 | 이유식 시작, 우유 전환, 배변훈련 | **높음 — 즉시 구매로 연결** |

---

### L3. 케어 루틴 전환점 상세 (추가된 기준)

#### 🍼 식이 전환 (가장 강력한 구매 트리거)

| 전환 시점 | 월령 | 기준 | 전환 내용 | 큐레이션 제품군 |
|---|---|---|---|---|
| **이유식 초기** | 4~6개월 | 대한소아과학회·WHO | 모유/분유 → 쌀미음·채소 퓨레 병행 시작 | 이유식 조리기, 냉동 큐브 트레이, 실리콘 턱받이, 소형 식판 |
| **이유식 중기** | 7~9개월 | 대한소아과학회 | 다진 식감, 단백질(육류·두부) 추가 | 이유식 분쇄기, 핸드블렌더, 유아용 수저 세트 |
| **이유식 후기** | 10~12개월 | 대한소아과학회 | 으깬 식감, 손가락 음식(finger food) | 흡착 식판, 자기 주도 이유식 도구, 비비백 |
| **완료기·유아식 전환** | 12~18개월 | 대한소아과학회 | 분유 → **생우유** 전환, 가족 식사로 점진 합류 | 유아용 컵(빨대컵→열린 컵), 유아 수저, 보조 식탁 의자 업그레이드 |
| **유아식 완성** | 24개월+ | 영양 가이드라인 | 가족 식사 완전 합류, 간식 루틴 정착 | 도시락통, 유아 간식 큐레이션, 물병 |

#### 💤 수면 루틴 전환

| 전환 시점 | 월령 | 전환 내용 | 큐레이션 제품군 |
|---|---|---|---|
| **수면 교육 시작** | 4~6개월 | 낮밤 구분, 수면 루틴 형성 가능 시기 | 수면 유도 백색소음기, 수면 자루(sleeping sack) |
| **낮잠 통합** | 12~18개월 | 낮잠 2회 → 1회로 통합 | 낮잠 매트, 암막 커튼 |
| **낮잠 졸업** | 36개월+ | 낮잠 없이 야간 수면으로 통합 | 싱글 침대 전환, 아동용 베개·이불 |

#### 🚽 위생·자립 루틴 전환

| 전환 시점 | 월령 | 전환 내용 | 큐레이션 제품군 |
|---|---|---|---|
| **배변훈련 시작** | 18~24개월 | 신체 신호 인지 시작 — 변기 도입 | 유아 변기·변기 시트, 팬티형 기저귀, 보조 발판 |
| **기저귀 졸업** | 24~36개월 | 주간 기저귀 졸업 → 속옷 전환 | 유아 속옷, 방수 매트리스 커버 |
| **혼자 양치** | 24~36개월 | 스스로 양치 시도 시작 | 유아 전동칫솔, 모래시계 타이머 |
| **혼자 씻기** | 36개월+ | 스스로 손 씻기·샤워 보조 | 아동 전용 스텝 스툴, 유아 샴푸 |

#### 👟 이동·신체 활동 전환

| 전환 시점 | 월령 | 전환 내용 | 큐레이션 제품군 |
|---|---|---|---|
| **첫 신발** | 10~13개월 | 붙잡고 서기 → 걷기 준비 | 보행화(발볼 넓은 유아화) |
| **유모차 → 킥보드** | 24~36개월 | 스스로 이동 욕구 증가 | 킥보드(2~3세용), 헬멧·보호대 |
| **세발자전거·자전거** | 36~48개월 | 균형감각 발달 | 세발자전거, 균형 자전거(밸런스 바이크) |

### 권장 큐레이션 구간 (7단계) — L1+L2+L3 통합

| 구간 | 월령 | L2 발달 키워드 | L3 케어 루틴 전환점 | 핵심 구매 트리거 |
|---|---|---|---|---|
| **0단계: 신생아** | 0~1개월 | 적응·수면·애착 | 완전 수유기 (모유/분유) | 수유용품, 신생아 의류, 침구, 스킨케어 |
| **1단계: 첫 웃음** | 2~4개월 | 오감 자극, 목 가누기 | 수면 루틴 형성 시작 | 모빌, 바운서, 수면 자루, 백색소음기 |
| **2단계: 뒤집기** | 5~8개월 | 뒤집기·앉기 | **이유식 시작 (초기·중기)** ← 핵심 전환 | 이유식 조리기, 식판, 수저, 턱받이 |
| **3단계: 기기** | 9~12개월 | 기기·붙잡고 서기 | 이유식 후기, 손가락 음식 | 흡착 식판, 안전 게이트, 바닥 매트 |
| **4단계: 첫 걸음** | 13~18개월 | 걷기, 언어 폭발 | **분유→우유 전환, 첫 신발** ← 핵심 전환 | 유아용 컵, 보행화, 낮잠 통합 용품 |
| **5단계: 뛰기** | 19~36개월 | 달리기, 2~3단어 문장 | **배변훈련, 기저귀 졸업** ← 핵심 전환 | 변기, 팬티형 기저귀, 속옷, 킥보드 |
| **6단계: 상상놀이** | 37~60개월 | 상상력·사회성·취학 준비 | 낮잠 졸업, 혼자 씻기 | 균형 자전거, 학습교구, 유아 책상 |

### 설계 원칙

1. **L3 전환을 "알림 이벤트"로 설계**: "이유식 시작했어요", "우유로 넘어갔어요", "배변훈련 시작했어요" 체크 시 해당 제품군 즉시 노출
2. **건강검진 구간과 정렬**: 9개월·18개월·30개월 검진 시점을 단계 경계로 — 부모의 기억 앵커
3. **±1단계 허용**: 발달 개인차 반영, 현재 단계 ±1 제품도 함께 노출
4. **안전인증 배지 필수**: KC인증·식품안전(이유식 도구) 필터 기본 ON
5. **L3 전환은 월령보다 행동 기준 우선**: "6개월이 됐어요"보다 "이유식 시작했어요"가 더 정확한 트리거

## 4. 발달 장난감·플레이키트 큐레이션 기준

### 글로벌 벤치마크: Lovevery (피카비류 플레이키트의 원형)

**Lovevery**는 현재 가장 과학적으로 정교한 플레이키트 서비스. 2~3개월마다 발달 단계에 맞는 키트를 구독 형태로 제공.

| 항목 | 내용 |
|---|---|
| 전문가 팀 | 아동발달 심리학자, 몬테소리 전문가, 작업치료사, 소아과 의사 공동 설계 |
| 근거 | 출판된 아동발달 연구 논문 기반 + 전문가 검토 |
| 교육 철학 | 몬테소리 + 발도르프(Waldorf) 혼합 |
| 구성 방식 | 단계별 Play Guide 포함 — "왜 이 장난감인가", "어떻게 놀아야 하는가", "지금 어떤 능력이 발달 중인가" 설명 |

---

### 발달 장난감 큐레이션의 근거 프레임워크 (혼합 적용 권장)

#### ① AAP "Power of Play" (2018) — 가장 공신력 있는 의학적 근거

- 놀이는 사회정서·인지·언어·자기조절 능력을 동시에 발달시키는 핵심 수단
- **개방형(open-ended) 장난감**이 창의성·집중력·문제해결력 발달에 우월 (블록, 인형, 미술도구)
- 빛이 번쩍이고 소리가 자동으로 나는 전자 장난감은 아이의 주도적 탐색을 방해함 → **수동형 장난감 우선** 원칙
- 출처: [AAP Pediatrics 2018 — The Power of Play](https://publications.aap.org/pediatrics/article/142/3/e20182058/38649/The-Power-of-Play-A-Pediatric-Role-in-Enhancing)

#### ② 몬테소리(Montessori) 원칙

- 아이의 발달 민감기(Sensitive Period)에 맞는 소재와 활동 제공
- 단순하고 자연 소재(나무, 천, 금속)로 만들어진 장난감 권장
- 한 번에 소수의 장난감만 제공 — 선택 과잉이 집중력을 방해함 → **큐레이션이 많을수록 좋다**는 플랫폼 논리와 정면 충돌하는 포인트 (플랫폼 설계 시 "추천 3~5개 핵심 픽" 방식 고려)
- 출처: [Lovevery CEO — Montessori/Reggio 비교](https://blog.lovevery.com/learning-cognitive-skills/rie-montessori-reggio-emilia-more-choosing-the-best-educational-approach-for-your-child-by-lovevery-ceo-jessica-rolph/)

#### ③ 픽클러(Pikler) 원칙 — 운동발달에 특화

- 아이가 스스로 자연스럽게 운동을 터득하도록 충분한 시간·공간·안전 제공
- 보행기·점퍼루 등 인위적으로 발달을 "앞당기는" 도구는 권장하지 않음
- 장난감보다 **안전한 바닥 환경** (매트, 픽클러 트라이앵글 등)이 더 중요한 단계 있음

#### ④ Dr. Toy 원칙 (Auerbach) — 장난감 3가지 가치

| 가치 | 내용 | 예시 |
|---|---|---|
| Active Value (활동성) | 신체 움직임을 유도 | 공, 매트, 트램펄린 |
| Creative Value (창의성) | 아이가 결과를 만들어냄 | 블록, 점토, 크레용 |
| Educational Value (교육성) | 인지·언어·수 개념 발달 | 퍼즐, 소팅 장난감, 그림책 |

### 월령별 발달 장난감 큐레이션 상세

| 구간 | 월령 | 발달 포커스 | 핵심 장난감 | 피해야 할 것 |
|---|---|---|---|---|
| **신생아** | 0~2개월 | 시각 자극 (흑백 대비), 청각 | 흑백 모빌, 딸랑이, 패브릭 북 | 과도한 소리/빛 자극 전자 장난감 |
| **첫 웃음** | 3~5개월 | 색상 인식, 손 뻗기, 잡기 | 컬러 모빌, 손목 딸랑이, 배밀이 매트 | 보행기(픽클러 원칙상 비권장) |
| **앉기** | 6~8개월 | 양손 협응, 원인-결과 인지 | 쌓기 링, 소프트 블록, 팝업 장난감, 거울 장난감 | 자동으로 움직이는 전자 장난감 |
| **기기** | 9~12개월 | 손가락 집기(pincer grasp), 대상영속성 | 소팅 박스(도형 끼우기), 컵 쌓기, 당기기 장난감, 첫 퍼즐 (2~3조각) | 너무 작은 조각 (질식 위험) |
| **걷기** | 13~18개월 | 대근육 이동, 인과관계, 모방 | 밀고당기기 장난감, 두드리기 장난감, 소꿉놀이 첫 단계, 낙서 크레용 | 복잡한 규칙의 장난감 |
| **뛰기** | 19~36개월 | 역할극, 언어 폭발, 상상력 시작 | 역할극 소품(주방·의사), 블록·레고 듀플로, 점토/모래, 퍼즐(6~12조각) | 완성품 장난감 (아이가 만들 여지 없음) |
| **상상놀이** | 37~60개월 | 규칙 이해, 협동, 읽기 전 단계 | 보드게임(단순 규칙), 미술도구 세트, 과학 탐구 키트, 알파벳 블록 | 과도한 스크린 기반 장난감 |

## 5. 월령별 책 큐레이션 기준

### 근거: AAP + ZERO TO THREE 조기 문해력 프레임워크

AAP는 생후 6개월부터 소아과 방문 시 책 읽기를 권장하며, 조기 독서는 언어 발달·어휘·사회정서 발달 모두에 영향을 미침.

#### 조기 문해력 6대 핵심 역량 (Carnegie Library / ILA 기준)

| 역량 | 정의 | 책 큐레이션 적용 |
|---|---|---|
| **Print Motivation** | 책에 대한 흥미·즐거움 | 아이가 직접 만지고 탐색할 수 있는 책부터 |
| **Phonological Awareness** | 소리와 단어의 인식 | 라임(운율)·반복 문장이 있는 책 |
| **Vocabulary** | 어휘력 | 사물 이름이 명확한 그림책 |
| **Print Awareness** | 글자가 의미를 가진다는 인식 | 큰 글씨, 단순한 레이아웃의 책 |
| **Letter Knowledge** | 글자 모양 인식 | 36개월 이후 알파벳/한글 입문 책 |
| **Narrative Skills** | 이야기 구조 이해 | 기승전결이 있는 그림책 |

---

### 월령별 책 유형 큐레이션

| 구간 | 월령 | 발달 포커스 | 책 유형 | 예시 특징 |
|---|---|---|---|---|
| **신생아~3개월** | 0~3개월 | 시각 자극, 엄마 목소리 인식 | 흑백 대비 그림책, 패브릭 북 | 텍스트 없음 또는 1~2단어, 고대비 이미지 |
| **4~6개월** | 4~6개월 | 색상 인식, 청각 자극 | 컬러 보드북, 소리나는 책 | 밝은 원색 그림, 의성어 반복 |
| **7~12개월** | 7~12개월 | 사물 인식, 가리키기 | 사물 그림책, 촉감책 | "이게 뭐야?" 질문 유도 가능한 구조, 두꺼운 보드북 |
| **13~24개월** | 13~24개월 | 언어 폭발, 모방 | 운율·반복 그림책, 일상 이야기 | 짧은 문장 반복, 라임, 행동 따라하기 유도 |
| **25~36개월** | 25~36개월 | 이야기 구조 이해, 감정 인식 | 감정 그림책, 기승전결 그림책 | 주인공의 감정·상황이 명확한 스토리 |
| **37~48개월** | 37~48개월 | 인과관계, 상상력, 사회성 | 창작 그림책, 논픽션 입문 | "왜?", "만약에?" 질문을 유도하는 내용 |
| **49~60개월** | 49~60개월 | 읽기 전 단계(pre-reading) | 한글/영어 입문책, 시리즈 그림책 | 반복 패턴으로 글자 예측 가능한 구조 |

---

### 장난감·책 큐레이션 설계 원칙 (플랫폼 적용)

1. **"왜 이 제품인가" 설명 필수**: Lovevery처럼 제품 옆에 발달 근거 1~2줄 표시 → 단순 쇼핑이 아닌 교육 경험
2. **핵심 픽 3~5개 제한**: 몬테소리 원칙 — 선택지가 많으면 오히려 부모가 지침. 단계별 "이번 달 필수 3개" 포맷
3. **개방형 장난감 우선 배지**: AAP 권장 기준 — 빛/소리 자동작동 전자 장난감은 별도 표시
4. **책+장난감 세트 큐레이션**: 같은 주제(예: 동물) 책과 장난감을 묶어서 추천 → 연계 놀이 유도
5. **발달 영역 태그**: 대근육/소근육/언어/인지/사회정서 — 각 제품에 태그 부착해 부모가 원하는 영역 필터링 가능

### 1. 핵심 스탠다드 4-Layer — 선정 이유

| Layer | 기준 | 출처 | 선정 이유 | 플랫폼에서의 역할 |
|---|---|---|---|---|
| **L1** | 의료·검진 기준 | 한국 영유아 건강검진 (국민건강보험공단) + AAP Well-Child Visit | 한국 부모가 가장 익숙한 공신력 있는 기준. "검진 때 뭐 사야 하지?" 패턴과 직결 | 구간 경계 설정 + 신뢰도 근거 |
| **L2** | 발달 행동 기준 | WHO 운동발달 + Piaget 인지발달 + Erikson 심리사회 발달 | 아이가 "무엇을 할 수 있게 되는가"를 설명. 부모가 직접 관찰 가능한 행동 기반 → 공감과 맥락 제공 | 제품 추천의 스토리 레이어 |
| **L3** | 케어 루틴 전환 | 대한소아과학회 식이 가이드라인 + WHO 영양 기준 | 이유식·우유 전환·배변훈련 등 **직접 구매로 이어지는 가장 강력한 트리거**. 월령보다 행동 이벤트 기반 | 핵심 구매 트리거 |
| **L4** | 발달 자극 기준 | AAP Power of Play (2018) + 몬테소리 + 픽클러 | 장난감·책을 "왜 이 시기에 이것인가"로 설명하는 근거. Lovevery·피카비의 차별화 포인트이자 이 플랫폼이 단순 쇼핑몰과 달라지는 핵심 | 장난감·책 큐레이션 근거 |

### 2. 7단계 구간 정의

구간 경계는 **한국 건강검진 시점 + WHO 운동발달 마일스톤**을 기준으로 정렬.  
부모 기억에 가장 강하게 각인되는 사건(첫 걸음, 이유식 시작 등)을 구간명으로 사용.

| 단계 | 구간명 | 월령 | 구간 설정 근거 |
|---|---|---|---|
| **Stage 0** | 신생아 | 0~1개월 | 출생~첫 검진(14일). 완전 수유기 |
| **Stage 1** | 첫 웃음 | 2~4개월 | 4개월 건강검진. 사회적 미소 시작 |
| **Stage 2** | 뒤집기 | 5~8개월 | 이유식 시작(4~6개월)·앉기(6개월)·WHO 운동발달 기준 |
| **Stage 3** | 기기 | 9~12개월 | 9개월 건강검진. WHO 기기~걷기 준비 구간 |
| **Stage 4** | 첫 걸음 | 13~18개월 | 18개월 건강검진. 보행 안정화 + 분유→우유 전환 |
| **Stage 5** | 뛰기 | 19~36개월 | 30개월 건강검진. 언어 폭발 + 배변훈련 구간 |
| **Stage 6** | 상상놀이 | 37~60개월 | 42·54개월 검진. Piaget 전조작기 전성기 |

### 3. 단계별 통합 매트릭스

#### Stage 0 — 신생아 (0~1개월)

| **L2 발달** | 반사 행동(빨기·잡기·모로반사) / 흑백 대비 시각 / 부모 목소리 인식 / 하루 16~18시간 수면 |

#### Stage 1 — 첫 웃음 (2~4개월)

| Layer | 항목 |
|---|---|
| **L1 검진** | 4개월 건강검진 — 대근육 운동(목 가누기), 사회성(눈 맞춤·미소) |
| **L2 발달** | 사회적 미소 시작 / 목 가누기(3~4개월) / 옹알이 시작 / 손 뻗기·잡기 시도 / 색상 인식 향상 |

#### Stage 2 — 뒤집기 (5~8개월)

| Layer | 항목 |
|---|---|
| **L1 검진** | 이 구간 공식 검진 없음 (다음: 9개월). 소아과 이유식 상담 권장 |
| **L2 발달** | 뒤집기(5~6개월) / 혼자 앉기 시도(6~7개월) / 양손 협응 / 원인-결과 인지(떨어뜨리기 반복) / 낯가림 시작 |
| **L3 케어** | **⭐ 이유식 시작 (4~6개월, 대한소아과학회)** / 초기: 쌀미음·단일 채소 퓨레 / 중기(7~8개월): 다진 식감, 단백질 추가 |
| **L4 자극** | 쌓기 링 / 소프트 블록 / 거울 장난감(자기 인식) / 팝업 장난감 / 촉감책·사물 그림책 |
| **핵심 전환점** | ⭐ **이유식 시작** — 가장 강력한 단일 구매 트리거 |
| **구매 트리거** | 이유식 조리기·핸드블렌더·냉동 큐브 / 이유식 식판·수저 세트 / 실리콘 턱받이 / 보조 식탁 의자 / 안전 게이트(뒤집기 이후 공간 확장) |

---

#### Stage 3 — 기기 (9~12개월)

| Layer | 항목 |
|---|---|
| **L1 검진** | 9개월 건강검진 — 대·소근육, 언어(옹알이), 사회성, 시각·청각 |
| **L2 발달** | 기기(9개월) / 붙잡고 서기(8~10개월) / 손가락 집기 pincer grasp(9~10개월) / 대상영속성 완성 / 첫 단어 시도("엄마","아빠") |
| **L3 케어** | 이유식 후기 — 으깬 식감·손가락 음식(finger food) 도입 / 수면 2회→1회 전환 시도 |
| **L4 자극** | 도형 소팅 박스 / 컵 쌓기 / 당기기 장난감 / 간단한 2~3조각 퍼즐 / 사물 이름 그림책 ("이게 뭐야?" 유도) |
| **핵심 전환점** | 기어다니기 시작 → 안전 환경 전면 재구성 필요 |
| **구매 트리거** | 안전 게이트·모서리 가드 / 바닥 매트 업그레이드 / 흡착 식판 / 손잡이 컵 / 치발기 (유치 출현) |

---

#### Stage 4 — 첫 걸음 (13~18개월)

| Layer | 항목 |
|---|---|
| **L1 검진** | 18개월 건강검진 — 보행, 언어(단어 수), 사회성, 자조(혼자 먹기) |
| **L2 발달** | 혼자 걷기 안정화 / 언어 폭발 시작(50단어 이상) / 모방 행동 증가 / 소꿉놀이 첫 시도 / 애착 대상(인형·담요) 형성 |
| **L3 케어** | **⭐ 분유→생우유 전환 (12개월)** / 유아식 시작 (가족 식사 합류) / 낮잠 2회→1회 통합 / 첫 신발 필요 |
| **L4 자극** | 밀고당기기 장난감 / 두드리기·끼우기 / 역할극 첫 소품(전화기·컵) / 낙서 크레용 / 운율·반복 그림책 / 일상 이야기 그림책 |
| **핵심 전환점** | ⭐ **분유 졸업 + 첫 신발** / 유아식 전환 |
| **구매 트리거** | 유아용 컵(빨대컵→열린 컵) / 유아 수저 업그레이드 / **보행화** / 낮잠 매트 / 역할극 첫 장난감 세트 |

---

#### Stage 5 — 뛰기 (19~36개월)

| Layer | 항목 |
|---|---|
| **L1 검진** | 30개월 건강검진 — 언어(문장), 사회성(또래), 인지, 대소변 가리기 |
| **L2 발달** | 달리기·계단 오르기 / 2~3단어 문장 → 4~5단어 문장 / 자기주장 강해짐("싫어") / 가작놀이(역할극) 본격 시작 / 또래 관심 시작 |
| **L3 케어** | **⭐ 배변훈련 시작 (18~24개월)** / **⭐ 기저귀 졸업 (24~36개월)** / 혼자 양치 시도 / 식사 스스로 먹기 안정화 |
| **L4 자극** | 역할극 소품(주방·의사·공구) / 듀플로·블록 / 점토·모래·물감 / 6~12조각 퍼즐 / 감정 그림책 / 기승전결 있는 그림책 |
| **핵심 전환점** | ⭐ **배변훈련 시작·기저귀 졸업** / 킥보드 입문 |
| **구매 트리거** | 유아 변기·변기 시트·발판 / 팬티형 기저귀 → 속옷 / 방수 매트리스 커버 / 킥보드+헬멧 세트 / 유아 전동칫솔 / 역할극 장난감 세트 |

---

#### Stage 6 — 상상놀이 (37~60개월)

| Layer | 항목 |
|---|---|
| **L1 검진** | 42개월 검진 (집중력·또래 관계) / 54개월 검진 (취학 전 발달 종합) |
| **L2 발달** | 상상력 기반 복잡한 역할극 / 규칙 이해·보드게임 가능 / 친구와 협동놀이 / 읽기 전 단계(글자 관심) / Piaget 전조작기 전성기 |
| **L3 케어** | 낮잠 졸업(36개월+) / 혼자 씻기 보조 / 취학 준비 루틴 형성 |
| **L4 자극** | 단순 규칙 보드게임 / 미술도구 세트 / 균형 자전거(밸런스 바이크) / 한글·숫자 입문 교구 / "왜?"를 유도하는 창작 그림책·논픽션 입문 |
| **핵심 전환점** | 취학 준비 / 세발자전거→자전거 / 싱글 침대 전환 |
| **구매 트리거** | 균형 자전거+헬멧 / 아동용 책상·의자 / 한글·수학 학습 교구 / 보드게임 / 아동용 침구(싱글 사이즈) |

---

### 4. 핵심 구매 트리거 TOP 8 (우선순위 순)

> 아래 트리거는 **부모가 플랫폼에서 직접 체크하는 이벤트**로 설계. 체크 즉시 해당 제품군 큐레이션 화면으로 전환.

| 순위 | 트리거 이벤트 | 해당 단계 | 예상 구매 제품군 |
|---|---|---|---|
| 1 | 이유식 시작했어요 | Stage 2 | 이유식 조리기·식판·수저·의자 풀세트 |
| 2 | 기어다니기 시작했어요 | Stage 3 | 안전 게이트·바닥 매트·모서리 가드 |
| 3 | 첫 걸음마 뗐어요 | Stage 4 | 보행화 |
| 4 | 분유 끊었어요 | Stage 4 | 유아 컵·수저·유아식 관련 식품 |
| 5 | 배변훈련 시작했어요 | Stage 5 | 유아 변기·팬티형 기저귀·발판 |
| 6 | 기저귀 완전히 뗐어요 | Stage 5 | 유아 속옷·방수 커버 |
| 7 | 아기 낳았어요 (D-day) | Stage 0 | 출산 준비 풀세트 |
| 8 | 첫 생일이에요 | Stage 3→4 전환 | 돌잔치 용품·첫 신발·유아식 용품 |

---

### 5. 설계 원칙

#### 구조 원칙
1. **월령보다 이벤트 우선**: "6개월이 됐어요"보다 "이유식 시작했어요"가 더 정확한 큐레이션 트리거. 앱 온보딩에서 월령 + 현재 상태(이유식 여부, 보행 여부 등) 동시 입력
2. **±1단계 노출**: 발달 개인차 반영 — 현재 단계 ±1 구간 제품도 함께 노출
3. **핵심 픽 3~5개 제한**: 몬테소리 원칙 기반. 단계별 "이번 달 필수 픽" 포맷으로 선택 피로 제거

#### 콘텐츠 원칙
4. **"왜 이 제품인가" 1줄 설명 필수**: Lovevery 방식. 발달 근거 없는 나열은 쿠팡과 동일. 예: *"지금 손가락 집기가 발달하는 시기예요 → 소팅 박스가 최적"*
5. **L4 장난감 배지 시스템**: AAP 기준 개방형 장난감 / 자동전자 장난감 / 몬테소리 권장 여부를 배지로 표시

#### 인증/신뢰 원칙
6. **안전인증 기본 ON**: KC인증(완구) / 식품안전인증(이유식 도구) / 유기농 인증(스킨케어) — 단계별 필수 인증 자동 필터
7. **AI 리뷰 요약**: 타사 리뷰 통합 + 월령별 사용 후기 분리 표시. "6개월 아기 엄마들의 후기"와 "12개월 아기 엄마들의 후기"를 따로 보여줌

#### 확장 원칙
8. **일본 진출 대응**: L1을 AAP 기준으로 병행 표기하면 일본 시장에서도 동일 프레임워크 적용 가능. 구간명은 행동 기반(첫 걸음, 뒤집기)이라 번역 후에도 직관적
