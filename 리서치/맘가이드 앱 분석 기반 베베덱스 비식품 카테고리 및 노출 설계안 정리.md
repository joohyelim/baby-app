# 베베덱스 비식품 카테고리 3단계 노출 구조

- 작성일: 2026-07-10
- 목적: 앱 노출 레이어 기준, 유저가 실제로 탐색하는 카테고리 구조 정의
- 구조: **1단계(홈 타일) → 2단계(칩 필터) → 3단계(세부 구분/필터 태그)**
- 관련 문서: 맘가이드 앱 분석 기반 비식품 카테고리 및 노출 설계안 (2026-07-10)

---

## 설계 전제

1. **3단계는 페이지 뎁스가 아니라 필터/태그로 처리한다.** 0~2세 카테고리 규모(품목 수) 대비 탭 3회 진입 구조는 과함. 유저 여정은 "타일 탭 → 칩으로 좁히기 → 필요 시 태그로 더 좁히기"의 2뎁스 + 필터 구조로 한다.
2. 미개방 차수(3·4차) 타일은 홈에 노출하되 '곧 열려요' 상태로 표기한다. 칩 단위 미개방(소독기, 치발기 등)은 해당 칩에만 '준비 중' 뱃지를 단다.
3. 3단계 세부 구분은 부모의 실제 검색 언어(밴드형/팬티형, PPSU/유리, 무기자차/유기자차 등) 기준으로 설계한다.

| ☀️ 진정·선케어 | 2차 | 전체 / 선케어 / 수딩젤 / 발진·태열크림 | 선케어: 무기자차·유기자차, 스틱·크림<br>발진·태열: 기저귀발진크림·태열크림 |

## 확정 필요 사항

1. **3단계 세부 구분 전체가 제안 상태.** 기획안 원문에 없는 부분으로, 부모의 검색 언어 기준으로 제안한 것. 특히 아래 3개는 구매 결정 직결 필터이므로 우선 확정 필요:
   - 기저귀 사이즈 체계 (NB·S·M·L·XL)
   - 젖병 재질 구분 (PPSU·유리·PP)
   - 체온계 방식 구분 (비접촉·귀·겨드랑이)
2. 물티슈 2단계 칩(일반형/캡형/휴대용)도 제안 상태 — 원문에는 물티슈 단독 타일만 정의됨
3. 유아 클리너(장난감클리너 등)의 노출 위치: 원문 표에 매핑 없음. 별도 칩 처리 또는 🫧 유아 빨래 타일 편입 여부 결정 필요
