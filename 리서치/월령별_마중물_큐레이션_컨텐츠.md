## 표기 규칙

- **필수도**: ⭐ 꼭 필요 (core) / ◽ 있으면 편함 (optional)
- **🔔 트리거**: 월령보다 우선하는 행동 기반 노출 조건 (예: "이유식 시작했어요")
- **★ 핵심 전환**: 가장 강력한 구매 트리거가 되는 케어 루틴 전환점
- **도메인**: 식품(FOOD) · 식기·수유(WARE) · 교구·완구·책(TOY) · 위생·생활(LIFE) · 의류·침구(WEAR) · 이동·외출(GEAR) · 안전(SAFETY) · 수면(SLEEP) · 영양제·건강기능식품(SUPPLEMENT)

## 한눈에 보기 (7단계)

| # | 단계 | 월령 | 핵심 전환점 | 대표 템빨 |
|---|---|---|---|---|
| 0 | 신생아 | 0~1개월 | ★ 완전 수유기 | 분유, 젖병, 배냇저고리 |
| 1 | 첫 웃음 | 2~4개월 | 수면 루틴 형성 시작 | 분유, 컬러 모빌, 바운서 |
| 2 | 뒤집기 **(데모)** | 5~8개월 | ★ 이유식 시작 (초기·중기) | 초기 이유식, 이유식 재료, 분유 |
| 3 | 기기 | 9~12개월 | ★ 이유식 후기 · 손가락 음식(finger food) | 후기 이유식, 분유, 흡착 식판 |
| 4 | 첫 걸음 | 13~18개월 | ★ 분유 → 생우유 전환·유아식 합류 | 생우유, 유아용 컵, 보행화 |
| 5 | 뛰기 | 19~36개월 | ★ 배변훈련 시작 → 기저귀 졸업 | 유아 변기, 방수 매트리스 커버, 킥보드 |
| 6 | 상상놀이 | 37~60개월 | 낮잠 졸업·싱글 침대 전환 | 세발자전거, 싱글 침대, 아동 전용 스텝 스툴 |

> 위 표의 단계별 템빨 외에, 모든 단계에 **[전 단계 공통 · 상시 소모품]**(기저귀·물티슈·보습·세탁세제)이 함께 적용됩니다.

|---|---|---|---|---|
| 위생·생활 | 기저귀 | ⭐ | 매일 최다 사용하는 상시 소모품, 피부 직접 접촉. (배변훈련 후 점차 졸업) |  |

| 위생·생활 | 보습 로션·스킨케어 | ⭐ | 영유아 피부 장벽 보호를 위해 연중 상시 보습. |  |
| 위생·생활 | 아기 전용 세탁세제 | ◽ | 옷·침구 세탁에 상시 사용. 잔류·향료 저자극 필요. |  |

## 0단계 · 신생아 (0~1개월)

**발달 요약:** 환경 적응·수면·애착 형성 시기. 영양은 모유/분유에 전적으로 의존.

**발달 키워드:** 적응 · 수면 · 애착 · 체온 조절

**케어 루틴 전환점**

- 🍼 **식이 — 완전 수유기** ★ : 모유 또는 분유로만 영양 섭취. 하루 8~12회 수유.

**핵심 아이템 (템빨)**

| 도메인 | 아이템 | 필수도 | 왜 필요한가 | 🔔 트리거 |
|---|---|---|---|---|
| 식품 | 분유 | ⭐ | 모유 대체·보충 영양원. 이 시기 영양의 전부. |  |
| 식기·수유 | 젖병·젖병솔·젖병세정제 | ⭐ | 수유와 위생 세척 필수 도구. |  |
| 식기·수유 | 젖병소독기 | ◽ | 면역 약한 신생아 위생 보조. |  |
| 의류·침구 | 배냇저고리·속싸개·손싸개·모자 | ⭐ | 체온 조절 미숙 + 태열 케어. 피부에 종일 닿는 품목. |  |
| 의류·침구 | 가제 손수건·면 침구/이불 | ⭐ | 수유·목욕·태열 관리에 상시 사용. 순면 소재 권장. |  |
| 위생·생활 | 엉덩이 발진크림·태열 진정 크림 | ⭐ | 신생아 피부 장벽 보호·태열 관리. |  |
| 위생·생활 | 아기 워시/샴푸 | ⭐ | 목욕 세정. 저자극 성분 필요. |  |
| 위생·생활 | 아기욕조 | ◽ | 안전한 목욕 환경. |  |
| 안전 | 체온계 | ⭐ | 발열 모니터링 필수 도구(KC). |  |
| 안전 | 콧물흡입기·습도계 | ◽ | 호흡기·온습도 관리 보조. |  |
| 교구·완구·책 | 흑백 모빌 | ◽ | 시각 자극(흑백 대비)에 반응하는 시기. |  |
| 수면 | 수면 자루·백색소음기 | ◽ | 수면 환경 조성 보조. |  |
| 영양제·건강기능식품 | 비타민D (400 IU/일) | ◽ | 모유·혼합수유아 출생 직후~12개월 보충 권고(대한소아청소년과학회·AAP). 분유 1L/일↑면 불필요. | 모유·혼합수유 중 |

## 1단계 · 첫 웃음 (2~4개월)

**발달 요약:** 오감 자극에 반응하고 목을 가누기 시작. 색을 인식하고 손을 뻗어 잡으려 함.

**발달 키워드:** 오감 자극 · 목 가누기 · 색상 인식 · 손 뻗기

**케어 루틴 전환점**

- 💤 **수면 — 수면 루틴 형성 시작** : 낮밤 구분이 가능해지는 시기 — 수면 교육 시작 가능.

**핵심 아이템 (템빨)**

| 도메인 | 아이템 | 필수도 | 왜 필요한가 | 🔔 트리거 |
|---|---|---|---|---|
| 식품 | 분유 | ⭐ | 이 시기 영양의 중심은 여전히 모유·분유. (수유 지속) |  |
| 교구·완구·책 | 컬러 모빌·손목 딸랑이 | ⭐ | 색상 인식·손 뻗기 발달 자극. |  |
| 교구·완구·책 | 바운서·배밀이 매트 | ◽ | 목 가누기·몸 움직임 지지. |  |
| 식기·수유 | 치발기 | ◽ | 4~5개월 침 분비 증가·구강 탐색 시작. | 침을 많이 흘려요 |
| 수면 | 수면 자루·백색소음기 | ◽ | 수면 루틴 형성 보조. |  |
| 교구·완구·책 | 컬러 보드북·소리나는 책 | ◽ | 청각·시각 자극, 의성어 반복. |  |
| 영양제·건강기능식품 | 비타민D (400 IU/일) | ◽ | 모유·혼합수유아 12개월까지 보충 지속 권고(대한소아청소년과학회·AAP). | 모유·혼합수유 중 |

## 2단계 · 뒤집기 (5~8개월) ← 6개월 데모 구간

**발달 요약:** 뒤집고 혼자 앉기 시작. 양손 협응과 원인-결과를 인지. ★이유식을 시작하는 가장 강력한 구매 전환 시기.

**발달 키워드:** 뒤집기 · 혼자 앉기 · 양손 협응 · 원인-결과 인지

**케어 루틴 전환점**

- 🍼 **식이 — 이유식 시작 (초기·중기)** ★ : 모유/분유 → 쌀미음·채소 퓨레 병행 시작. 6~9개월은 이유식 초중기.

**핵심 아이템 (템빨)**

| 도메인 | 아이템 | 필수도 | 왜 필요한가 | 🔔 트리거 |
|---|---|---|---|---|
| 식품 | 초기 이유식 (쌀미음·채소 퓨레)·시판 이유식 | ⭐ | 이유식 시작 시기. 첫 고형식 도입. | 이유식 시작했어요 |
| 식품 | 이유식 재료 (쌀·채소·단백질) | ⭐ | 홈메이드 이유식용 식재료. | 이유식 시작했어요 |
| 식품 | 분유 | ⭐ | 이유식은 보충식 — 12개월 전까지 모유·분유가 영양 중심(하루 400ml 전후). |  |
| 식기·수유 | 이유식 조리기·냉동 큐브 트레이 | ⭐ | 이유식 조리·소분 보관 도구. | 이유식 시작했어요 |
| 식기·수유 | 소형 식판·유아 수저·실리콘 턱받이 | ⭐ | 이유식 먹이기 식기. 자기주도 식사 준비. | 이유식 시작했어요 |
| 식기·수유 | 빨대컵·치발기 | ◽ | 수분 섭취 연습·구강 발달. |  |
| 교구·완구·책 | 쌓기 링·소프트 블록·팝업 장난감·거울 장난감 | ◽ | 양손 협응·원인-결과 인지 자극. |  |
| 교구·완구·책 | 컬러 보드북·촉감책 | ◽ | 사물 인식·촉각 자극. |  |
| 영양제·건강기능식품 | 비타민D (400 IU/일) | ◽ | 모유·혼합수유아 12개월까지 보충 지속 권고(대한소아청소년과학회·AAP). | 모유·혼합수유 중 |
| 영양제·건강기능식품 | 철분 (1 mg/kg/일) | ◽ | 모유수유아 4~6개월부터 철분강화 이유식 전까지 보충 권고. 분유수유아는 불필요. | 모유수유 중 |
| 영양제·건강기능식품 | 유아 유산균(프로바이오틱스) | ◽ | 건강한 만삭아 일상 보충은 권장되지 않음(AAP 2024) — 선택·개별 상담. |  |

## 3단계 · 기기 (9~12개월)

**발달 요약:** 기어다니고 붙잡고 서기 시작. 손가락으로 집기(pincer grasp)와 대상영속성 발달. 손가락 음식 시작.

**발달 키워드:** 기기 · 붙잡고 서기 · 손가락 집기 · 대상영속성

**케어 루틴 전환점**

- 🍼 **식이 — 이유식 후기 · 손가락 음식(finger food)** ★ : 으깬 식감 + 스스로 집어먹는 자기주도 이유식으로 전환.
- 👟 **이동·신체 — 기기 시작 — 안전 환경 필요** ★ : 바닥 이동 범위 확장으로 가정 내 안전 사고 위험 증가.

**핵심 아이템 (템빨)**

| 도메인 | 아이템 | 필수도 | 왜 필요한가 | 🔔 트리거 |
|---|---|---|---|---|
| 식품 | 후기 이유식·핑거푸드·유아간식 | ⭐ | 으깬 식감·자기주도 섭취 시기. | 손으로 집어먹어요 |
| 식품 | 분유 | ⭐ | 12개월 전까지 수유 지속 — 고형식↑ 수유↓. |  |
| 식기·수유 | 흡착 식판·오픈컵·자기주도 이유식 도구 | ⭐ | 엎지름 방지·스스로 먹기 연습. |  |
| 안전 | 안전 게이트·바닥 매트·모서리 가드 | ⭐ | 기기·붙잡고 서기 시작 → 낙상·충돌 방지. | 기기 시작했어요 |
| 교구·완구·책 | 소팅 박스·컵 쌓기·당기기 장난감·첫 퍼즐(2~3조각) | ◽ | 손가락 집기·대상영속성 발달. |  |
| 교구·완구·책 | 사물 그림책·촉감책 | ◽ | "이게 뭐야?" 가리키기 유도. |  |
| 영양제·건강기능식품 | 비타민D (400 IU/일) | ◽ | 모유·혼합수유아 12개월까지 보충 지속 권고(대한소아청소년과학회·AAP). | 모유·혼합수유 중 |
| 영양제·건강기능식품 | 철분 (1 mg/kg/일) | ◽ | 모유수유아 철분강화 이유식·육류 섭취 전까지 보충 권고. 충분 섭취 시 불필요. | 모유수유 중 |

## 4단계 · 첫 걸음 (13~18개월)

**발달 요약:** 혼자 걷기 시작하고 언어가 폭발적으로 성장. ★분유→우유 전환, 첫 신발 구매 전환 시기.

**발달 키워드:** 걷기 · 언어 폭발 · 모방

**케어 루틴 전환점**

- 🍼 **식이 — 분유 → 생우유 전환·유아식 합류** ★ : 분유 졸업, 가족 식사로 점진 합류. 빨대컵 → 오픈컵 전환.
- 👟 **이동·신체 — 첫 신발** ★ : 붙잡고 서기 → 걷기 → 첫 보행화 필요.
- 💤 **수면 — 낮잠 통합** : 낮잠 2회 → 1회로 통합되는 시기.

**핵심 아이템 (템빨)**

| 도메인 | 아이템 | 필수도 | 왜 필요한가 | 🔔 트리거 |
|---|---|---|---|---|
| 식품 | 생우유·유아식 | ⭐ | 분유 졸업, 가족 식사 합류. | 우유로 넘어갔어요 |
| 식기·수유 | 유아용 컵(빨대컵→오픈컵)·유아 수저·보조 식탁 의자 | ⭐ | 자기주도 식사 도구 업그레이드. |  |
| 이동·외출 | 보행화 (발볼 넓은 유아화) | ⭐ | 첫 걸음 시작 — 발 보호. | 걷기 시작했어요 |
| 수면 | 낮잠 매트·암막 커튼 | ◽ | 낮잠 통합 시기 수면 환경. |  |
| 교구·완구·책 | 밀고당기기 장난감·두드리기 장난감·소꿉놀이 첫 단계·낙서 크레용 | ◽ | 대근육 이동·모방·인과관계 학습. |  |

## 5단계 · 뛰기 (19~36개월)

## 영양제(건강기능식품) 큐레이션 기준

> 영양제는 식품(FOOD)과 규제 트랙이 다른 **건강기능식품**입니다. 두 축으로 평가합니다.

**① 영양 권고 (왜/언제/얼마)**

- 근거: 대한소아청소년과학회 2021 소아청소년 영양 가이드라인 + AAP(미국소아과학회) 정책성명
- 비타민D: 400 IU/일 — 모유·혼합수유아 출생 직후~12개월 (분유 1L/일 이상이면 불필요)
- 철분: 1 mg/kg/일 — 모유수유아 4~6개월부터 철분강화 이유식 전까지 (분유수유아 불필요)
- 유산균(프로바이오틱스): 건강한 만삭아 일상 보충 권장 안 함(AAP 2024) → 선택·개별 상담

**② 제품 안심 (안전)**

- 관할: 식약처 「건강기능식품에 관한 법률」 (식품위생법 FOOD와 별도 트랙)
- 확인: 건강기능식품 인증마크, 고시형/개별인정형 기능성 원료, 영유아 섭취 적합 여부
- 표시: 일일섭취량·섭취 시 주의사항·이상사례 표시 준수
