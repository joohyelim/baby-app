# 0~2세 안심 제품 판별 앱 통합 자료집 (출처 포함)

> 영유아(0~2세) 제품을 **성분과 소재, 안전 인증**으로 읽어 "먹여도/써도 괜찮은지"를 ✓(안심특징), ✕(걱정요소)로 보여주는 앱의 기획과 구현 자료를 한 문서로 모았습니다.
> 각 섹션 끝에 **근거 출처(공식 기관과 법령)** 를 달았습니다.
> 작성 기준일: 2026-06-21

## 1. 한눈에 보는 전체 그림

```
제품 카드 예시                          🟢 안심 88
──────────────────────────────────────────────
 ① 성분 안심   →  원재료명, 전성분, 재질을 읽음 (해당 품목)
                  ✓ 무첨가 …      ✕ 걱정성분 …
 ② 신뢰 지표   →  인증, 원산지, 리콜을 읽음 (전 품목 공통)
                  ✓ KC   ◎ 제조국   ✓ 리콜 없음
```

- **대상:** 0~2세 영유아 제품. 1차 출시는 **조리원 퇴소 후 신생아 홈케어 필수품**부터.
- **카테고리:** 식품(FOOD), 식기/유아용품(WARE), 교구/완구(TOY), 위생/생활(LIFE) 4종에 기구와 의류(신뢰 지표)를 더함.
- **차별점:** ① 카테고리 통합(같은 ✓/✕ 언어) ② 인증과 성분 동시 ③ 발달 단계 큐레이션.

## 2. 평가 모델: 두 가지 기준

모든 제품을 하나의 카드 틀에 담기 위해 평가를 **두 갈래**로 나눕니다.

| 기준 | 무엇을 읽나 | 적용 품목 | 출력 |
|---|---|---|---|
| ① **성분 안심점수** | 원재료명, 전성분, 재질 텍스트 | 식품, 화장품, 생활화학, 식기 | ✓ 무첨가 / ✕ 걱정성분 + 점수 |
| ② **신뢰 지표** | 인증, 제조국, 브랜드, 리콜, 소재 | **전 품목 공통** | ✓ KC / ◎ 제조국 / 🔴 리콜 |

> 핵심 원칙: "비대상"은 평가가 불가능한 게 아니라, 평가 기준이 성분에서 신뢰 지표로 바뀌는 것입니다. 소독기, 체온계, 의류도 KC, 전기안전, 원산지, 리콜 같은 정량 신호로 평가할 수 있습니다.

**안심점수 산식 (초기값)**

- 감점: 고위험 성분 −25, 중위험 −15, 경위험 −7
- 가점: 핵심 인증 +10, 일반 안심배지 +3
- 등급: 90점 이상 🟢안심 / 70~89 🟡양호 / 50~69 🟠주의 / 50점 미만 🔴경고
- 신뢰도 게이트: 전성분 미표기 시 "✓ 무첨가"를 부여하지 않음(=정보 없음을 안심으로 착각 금지)

## 4. 카테고리별 인증 기준

| 카테고리 | 핵심 인증/제도 | 필수 게이트 | 가점(안심) |
|---|---|---|---|
| **FOOD** | 어린이 기호식품 품질인증, HACCP, (분유)표시기준 적합 | 식품 표시기준 준수 | 품질인증, 유기농, HACCP |
| **WARE** | 식품용 기구·용기 기준 적합(식품용 표시), (+젖병류 KC) | 식품용 적합 | 용출시험 통과, BPA-free, KC |
| **TOY** | KC(안전인증/안전확인/공급자적합성확인) | KC 보유 | 유해물질 시험, 친환경 도료 |
| **LIFE** | 위생용품 적합 / 안전확인대상 생활화학제품 신고 / (물티슈)화장품 | 신고, 기준 적합 | 무유해보존제, 친환경 인증 |

- 안전인증(가장 엄격, 제3자 인증) → 안전확인(시험성적서 신고) → 공급자적합성확인(자체 시험, 확인)

## 5. 카테고리별 안심특징과 걱정요소

### FOOD (식품)

| 구분 | 항목 |
|---|---|
| ✓ 안심 | 무합성보존료, 무인공색소, 무합성향료, 무MSG, 무합성감미료, 무발색제, 트랜스지방0, Non-GMO, 유기농, 품질인증, HACCP |
| ✕ 걱정 | 합성보존료(소르빈산, 안식향산), 인공색소(타르), 카라멜색소, 합성향료, 향미증진제(MSG), 합성감미료, 발색제(아질산나트륨), 경화유/트랜스, 당류와 나트륨 과다, 카페인, GMO 우려 |

| 0~2세 임계 | 1회 나트륨 50mg↓, 당류 3g↓, 재료 3~5개, 불명 첨가물 3개↑ 회피 |

## 6. 신뢰 지표로 보는 제품 (성분 비대상)

성분이나 소재의 독성만으로는 평가가 어려운 제품(젖병소독기, 아기욕조, 체온계, 콧물흡입기, 의류)도 출처와 인증, 소재 투명성으로 평가합니다.

### 공통 신뢰 지표 6종

| 지표 | 내용 | 신호화 |
|---|---|---|
| 안전 인증 | KC(어린이/전기), 식품용 적합, 위생용품, 친환경 | 보유=✓ / 미표시=⚠️ |
| 제조국/원산지 | 생산국, (해당 시) 브랜드 본국 | 정보 제공(가치판단 X) |
| 브랜드 신뢰도 | 운영 기간, AS, 인증 이력, 누적 평점 | 보조 신호 |
| 리콜, 안전사고 | 소비자24, 제품안전관리원 리콜 | 있음=🔴 |
| 소재/섬유 투명성 | 재질, 섬유 혼용률 표기 명확성 | 명확=✓ / 불명=⚠️ |
| 사용자 평가 | 동일 연령대 부모 리뷰, 재구매 | 보조 신호 |

### A. 기구와 전자 (젖병소독기, 욕조, 체온계, 흡입기)

- **이중 KC:** 전기제품(소독기, 흡입기)은 **전기용품과 어린이제품 이중 KC 안전인증** 대상입니다(전기용품 및 생활용품 안전관리법 §5 + 어린이제품법 §17).
- ✓: KC, 전기안전, 식품접촉부 BPA-free, 리콜 없음, 제조국과 AS 명확 / ⚠️: KC 미표시, 리콜 이력, 재질 불명.

### B. 의류와 섬유 (배냇저고리, 속싸개, 손싸개, 가제수건)

- **1차 범위:** 신생아 피부에 종일 닿는 속옷류만 신뢰 지표 품목으로 넣습니다(배냇저고리, 속싸개, 손싸개, 가제수건). 부모가 무형광, 무염색, 유해물질을 가장 많이 따지는 품목이라 신뢰 지표 모델과 잘 맞습니다. 일반 아동복(외출복, 시즌 의류)은 SKU가 많고 차별점이 약해 2차로 둡니다.
- **국내 법정:** 영유아 섬유제품은 어린이제품 안전특별법상 KC 대상입니다. 유아용은 안전확인, 아동용은 공급자적합성확인. 시험 항목은 **폼알데하이드, 아릴아민(아조염료), pH, 납, 카드뮴**.
- **국제 참고 인증:** **OEKO-TEX STANDARD 100**(특히 Product Class I = 36개월 이하 영유아용, 가장 엄격), **GOTS**(유기농 섬유).
- ✓: KC(안전확인/공급자적합성), OEKO-TEX Class I, GOTS, 순면 100%, 무형광, 유해물질 시험 통과, 원산지 명확.

## 8. 시작 스코프: 조리원 퇴소 후 신생아 홈케어

생후 약 2~4주, 집에서 바로 쓰는 제품부터. "가장 절실 + 매일 사용 + 성분 분석 가능 + 부모 걱정 큼" 기준.

### MVP 시작 5종

| 우선 | 제품 | 카테고리 | 자동화 | 비고 |
|---|---|---|---|---|
| ★1 | 기저귀 | LIFE(위생) | ★★★☆ | 매일 최다 사용, 피부 직접 접촉 |
| ★1 | 물티슈 | LIFE(화장품/위생) | ★★★☆ | 전성분 명확, 블랙리스트 판정 쉬움 |

| ★2 | 분유 | FOOD | ★★★★ | 공공 원재료 DB, 분유 전용 룰 |
| ★3 | 젖병 | WARE | ★★★☆ | 재질(BPA) 판정 |

### 분석 대상 vs 비대상

- ✅ 분석(성분이나 소재 텍스트 존재): 분유, 젖병, 젖병세정제, 기저귀, 물티슈, 발진크림, 워시/로션, 세탁세제
- ⬜ 신뢰 지표 중심(비대상): 젖병소독기, 세척솔, 아기욕조, 체온계, 콧물흡입기, 습도계
- ⬜ 신뢰 지표 중심(비대상, 신생아 피부 직접 닿는 속옷류): 배냇저고리, 속싸개, 손싸개, 가제수건
- △ 2차 확장: 일반 아동복(외출복, 우주복, 시즌 의류)

## 9. 구현 실험 결과 & 자동화 가능성

| 카테고리 | 데이터 확보 | 자동추출 | 막히는 지점 | 우선순위 |
|---|---|---|---|---|
| 식품(이유식, 간식) | 공공DB 양호 | ★★★★★ | 신상품, 수입품 지연 | **1순위** |
| 분유 | 양호 | ★★★★☆ | 전용 룰 분기 필요 | 1순위 |
| 식기, 젖병 | 표기 의존 | ★★★☆☆ | 재질 이미지, "기타수지" | 2순위 |
| 위생, 생활 | 전성분 텍스트 양호 | ★★★☆☆ | 분류 갈래, 신고 매칭 | 2순위 |
| 교구, 완구 | KC만 공개 | ★★☆☆☆ | 소재, 도료 비공개, 이미지 | 3순위 |

**핵심 발견**

- 분유는 "좋다/나쁘다"보다 **표시기준 적합 여부**가 핵심 → 일반 식품 룰과 분리한 **분유 전용 룰** 필요.
- 치발기처럼 입에 무는 교구는 법적 유해물질 기준이 젖병보다 느슨 → **젖병 기준으로 보수 적용**.
- 재질이 이미지로만 표기되거나 "기타 합성수지"로 뭉뚱그려진 경우가 많음 → **OCR + 제조사/공공DB 보강** 필요.
- 링크 단독 스크래핑은 ToS, 안티봇 위험 → **바코드 + 공공DB + OCR + 제휴** 하이브리드 권장.

## 10. 카드 UI 표시 모델

```
제품명                                  🟢 안심 88
──────────────────────────────────────────────
 ① 성분 안심  (해당 품목만 표시)
    ✓ 무파라벤   ✓ CMIT/MIT 무첨가   ✕ 향료
 ② 신뢰 지표  (전 품목 공통)
    ✓ KC 안전확인   ✓ OEKO-TEX Class I
    ◎ 제조국: 한국   ◎ 소재: 순면 100%
    ✓ 리콜 이력 없음
```

- 성분 분석되는 제품은 ①+②, 안 되는 기구나 의류는 ②만 채워도 카드 완성.
- "정보 없음"은 빈칸이 아니라 **"확인 필요/미표기"** 로 명시해 신뢰를 지킴.
- 원산지와 브랜드는 단정 대신 **투명 정보**로, 리콜이나 인증 미보유는 **명확한 경고**로 처리.
